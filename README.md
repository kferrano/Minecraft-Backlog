# Minecraft Master Backlog

This repository is the **central public backlog** for all of my Minecraft mods and modpacks.

It is used to:
- collect **bug reports** from players and admins
- collect **feature requests**
- track **todos, refactors, and technical debt**
- plan **releases, ports, and larger milestones**
- provide transparency for **modpack developers**

This repository does **not** contain mod source code.  
It is purely for tracking, planning, and coordination.
